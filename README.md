# Power BI Operations Dashboard

This repository demonstrates a beginner-friendly operational reporting dashboard built using Power BI concepts.

## Dashboard Metrics
- Total Enquiries
- Total Admissions
- Conversion Rate
- Monthly Performance Trend
- Enquiries by Source

## Tools Referenced
- Power BI
- Microsoft Excel / CSV
- Data Analysis
- Operational Reporting

## Use Cases
- Education Operations Reporting
- Business Performance Monitoring
- Monthly Trend Analysis
- Conversion Tracking
